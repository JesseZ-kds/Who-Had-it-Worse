# Who-had-it-Worse
A Webgame inspired by "Higher or Lower".

Greek Mythology is a key part of the game.
Comparison from Myth to Myth.

It's simple.
Watch both the **Ted Talks** given.  
Compare them and decide **"Who had it Worse?"**
